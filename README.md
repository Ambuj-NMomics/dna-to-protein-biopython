# DNA to Protein Converter using Biopython

## Overview
This project demonstrates fundamental bioinformatics operations using Biopython:

- DNA sequence object creation
- Transcription (DNA → mRNA)
- Translation (mRNA → Protein)
- Reverse complement generation
- Coding DNA Sequence (CDS) validation

## Technologies Used
- Python 3
- Biopython
- Linux (Conda environment)

## Installation

```bash
conda create -n bioinfo python=3.11
conda activate bioinfo
pip install biopython




#Run

python dna_to_protein.py
